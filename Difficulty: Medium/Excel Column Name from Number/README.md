<h2><a href="https://www.geeksforgeeks.org/problems/column-name-from-a-given-column-number4244/1?page=4&category=Strings,Java&sortBy=submissions">Excel Column Name from Number</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">Given a positive integer <strong>n</strong>, return its corresponding Excel column title. </span><span style="font-size: 18px;">In Excel, column titles follow this pattern:</span></p>
<p><span style="font-size: 18px;">A, B, C, ..., Z, AA, AB, ..., AZ, BA, ..., ZZ, AAA, AAB, ...</span></p>

<p><span style="font-size: 18px;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>n = 28
<strong>Output:</strong> AB
<strong>Explanation</strong>: 1 to 26 are A to Z, then 27 is AA and 28 is AB.</span>
</pre>
<pre><span style="font-size: 18px;"><strong>Input</strong>: n = 13
<strong>Output:</strong> M
<strong>Explanation</strong>: M is the 13th character of alphabet.</span>
</pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>n = 5473578
